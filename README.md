# 🚀 AI Resume Roaster & Instant Portfolio Builder

**Transforming boring PDFs into brutal roasts and deploy-ready portfolio websites in seconds.**

A lightweight, high-performance Micro-SaaS built for job seekers, tech enthusiasts, and developers. It uses cutting-edge AI to extract resume data, provide a ruthless (yet actionable) roast, and instantly generate a fully coded, premium dark-glassmorphism HTML/CSS portfolio website.

---

## 🔥 Key Features

*   **PDF Data Extraction:** Seamlessly parses text from uploaded resumes without storing sensitive user data on external databases.
*   **AI-Powered Roasting:** Leverages Gemini Pro's advanced NLP to analyze career gaps, weak bullet points, and formatting errors, delivering a brutal, engaging, and actionable review.
*   **1-Click Portfolio Generator:** Instantly writes a single-file HTML/CSS web code based on the user's exact resume data. Features a modern dark theme, neon accents, and glassmorphism UI.
*   **Zero-Database Architecture:** Operates entirely in-memory during the session, meaning zero server storage costs and infinite scalability.
*   **Premium Glass UI:** The frontend is injected with custom Base64 CSS rendering to break out of the standard Streamlit look, offering a sleek, dark-mode native experience.

---

## 💻 Tech Stack

*   **Frontend & Routing:** Streamlit (Python)
*   **AI Core Logic:** Google Gemini Pro API (LLM & Generative Code)
*   **PDF Processing:** PyPDF2
*   **Design Architecture:** Custom CSS Injection, Base64 Encoding, Glassmorphism UI
